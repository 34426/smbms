# 超市订单管理系统

ssm超市订单管理系统  ssm框架  java

主要功能有：订单管理，供应商管理，用户管理，密码修改等功能；
技术栈：spring，springmvc，mybatis，maven，tomcat，mysql，jsp等
推荐配置：idea，mysql8.0，tomcat8.0

三个用户角色：1、系统管理员 2、经理 3、普通员工
